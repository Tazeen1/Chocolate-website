# 🍫 ChocoaLove – Premium Chocolate Website

A modern and responsive chocolate website designed with a premium and elegant look. This project showcases different chocolate products with smooth UI and clean design.

## 🚀 Live Demo
https://tazeen1.github.io/chocolate-website/

## ✨ Features
- Elegant and premium UI design
- Hero section with visuals
- Product showcase (truffles, caramel bites, etc.)
- Image gallery
- Smooth scrolling and basic animations
- Fully responsive (mobile + desktop)

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript
